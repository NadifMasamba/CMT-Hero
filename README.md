This is a Telegram Bot written in Python for mirroring files on the Internet to your Google Drive or Telegram.


## Deploy on Heroku

* `HEROKU_EMAIL`
* `HEROKU_API_KEY`
* `HEROKU_APP_NAME`
* `CONFIG_FILE_URL`
